## ⚡ Sobre o projeto

O **J.A.R.V.I.S** é um assistente pessoal de inteligência artificial que roda localmente no seu computador.

Ele combina:

* 🎤 Comandos de voz naturais
* 👁️ Visão de tela e câmera
* 🧠 Memória persistente (até 2 meses)
* ⚡ Automação completa do sistema
* 🌐 Interface web local

Inspirado no sistema de Tony Stark, o objetivo é transformar seu PC em um assistente inteligente real.

---

## 🚀 Funcionalidades principais

* 🎙️ Voz masculina real (SAPI Windows)
* 🧠 IA com OpenAI + Groq (fallback automático)
* 👁️ Leitura de tela + análise de código
* 📷 Reconhecimento via câmera
* 🎵 Reprodução de músicas via YouTube
* 🌤️ Clima em tempo real
* 🖥️ Controle completo do sistema
* 🌐 Interface web interativa
* 💾 Sistema de memória inteligente
